# 100xmylife.ai

Local scaffold for the future `100xmylife.ai` website and messaging system.

## What this repo is

This repo is not the site build yet. It is a clean handoff package for Claude Code, containing:
- a lightweight website scaffold
- a curated `knowledge/` library for strategy, messaging, and offer design
- copied source material in `knowledge/raw/` for deeper mining

## Intended use

Open this repo in Claude Code and use the knowledge base to:
1. define positioning
2. draft site architecture and copy
3. build a GitHub Pages site in `site/` or `docs/`
4. preserve a durable context trail for future iterations

## Suggested build sequence

1. Read `knowledge/site-brief.md`
2. Read `knowledge/product-offer.md`
3. Read `knowledge/business-context.md`
4. Read `knowledge/human-in-the-loop.md`
5. Use `knowledge/source-index.md` and `knowledge/raw/` for proof, phrasing, and nuance

## Repo structure

- `knowledge/` curated and raw context for the site
- `site/` placeholder for the future website implementation

## Domain

Planned domain: **100xmylife.ai**
