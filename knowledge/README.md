# Knowledge Folder

This folder is the strategy pack for building the `100xmylife.ai` website.

## Structure

- `site-brief.md` — concise website strategy and page plan
- `brand-voice.md` — tone, writing principles, brand edges
- `business-context.md` — business model, clients, pricing, goals
- `product-offer.md` — what the service is and how it works
- `human-in-the-loop.md` — oversight, approvals, quality, trust
- `client-examples.md` — carefully framed proof points and case material
- `known-constraints.md` — risks, limitations, careful claims
- `100x-framework.md` — Nick's core leverage framework
- `source-index.md` — map of source files consulted
- `transcript-derived-insights.md` — synthesized insights from Fireflies and Limitless
- `positioning-and-packaging.md` — messaging, premium packaging, and what not to claim
- `operating-principles.md` — deeper truths shaping the product and service
- `raw/` — a safe local-source manifest that points Claude Code to the original workspace files without committing raw PII-heavy materials

## How to use it

Start with the curated docs. Use `raw/` to find the original local workspace files when you need deeper evidence or wording. Do not copy private details from those files into the repo without sanitizing them first.
